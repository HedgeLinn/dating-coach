## 使用方式

- **新增联系人**: 告诉我她的主页要求/标签 + 初始对话，自动编号建档
- **回复消息**: "回复 #001，她说……"，我会读画像 + 档案帮你写
- **新对话**: "开个新联系人"，启动建档流程
- **补充画像**: "补充画像：[新数据]"，同步更新 profile.md
- **更新进展**: 截图或描述互动，更新对应档案的进展日志
