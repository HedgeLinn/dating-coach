# ISTJ 实战卡片

> 聊之前扫一眼。不靠直觉，靠对照表。

---

## 一、反向翻译：她的要求 = 她的伤疤

| 她说的 | = 翻译 |
|--------|--------|
| 要专一 / 1v1 | = 被出轨过 |
| 要上进 | = 前任躺平 / 啃她 |
| 互相花钱花心思 | = 被白嫖过 |
| 不见家长 | = 被催婚 / 被前任家长否定 |
| 年龄无所谓 | = 她最在意年龄（姐弟恋=怕被嫌老） |
| 要真诚 / 别骗我 | = 被骗过 |
| 随时查岗 | = 前任撒谎/玩消失 |

---

## 二、消息分类：收到先判，再回

| 类型 | 怎么识别 | 怎么回 |
|------|---------|--------|
| 🔴 审查 | 她问具体信息（多大/做什么/有没有别人/照片） | 短答 + 诚实 + 不反问 |
| 🟡 调情 | 反话 / 啧 / 嘛 / 呢 / 逗你 / 「看来你……」「就这么不想……」「那你怕不怕」 | 接住调侃 + 不解释 + 不道歉 |
| 🟢 绿灯 | 主动分享 / 夸你 / 「哈哈哈哈」连着几个 / 反问你感受 | 回夸 + 推进（聊下去或约见面） |

---

## 三、沉默时：别追

```
她沉默了
├─ 刚发了审查回答？→ 等她消化。不追
├─ 刚发了调情？→ 等她起话题或你另起。不追
├─ 她刚暴露脆弱？→ 需要时间。不追（#001外形菜吗后离线40分钟就是）
├─ 超24h没动静？→ 起一个轻松新话题
└─ 你追「还在吗」？→ ❌ 追一次掉一档
```

---

## 四、该约了：三个灯全亮就关

- [ ] 她主动问过你个人信息（年龄/职业/感情经历）
- [ ] 她开过玩笑或逗过你（防线降了）
- [ ] 当前对话没有她等你的问题

→ 三个全亮 = 约。缺一个别急。

约法：咖啡厅/简餐/逛展。地点你定两个让她选，或直接你定。

---

## 五、你的武器（别忘）

| 优势 | 怎么用 |
|------|--------|
| 话少 | 不暴露需求感。不必每条都回一长串 |
| 诚实 | 审查阶段这是核武器。答就答真的 |
| 不吹牛 | "还在学"比"我很厉害"可信 |
| 有方向 | 周末咖啡厅+健身就是最好的自我介绍 |
| 外形过关 | 照片是你最好的被动筛选 |

---

## 六、禁区

- ❌ 追「还在吗」或连发两条
- ❌ 解释太多（一个回答超过三行就删）
- ❌ 「我对你这么好你为什么不喜欢我」（ISTJ 最容易犯）
- ❌ 假装自己是什么人设——你本色最值钱
- ❌ 第一次约会太高调/太贵
- ❌ 表白求爱——用行动升级，不用嘴
