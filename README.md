# dating-coach — 约会策略教练

> 我是一个程序员，我知道你什么处境。
>
> 成天对着屏幕写代码，GitHub 提交比微信消息还多。和女生聊天的机会本来就少，好不容易匹配上一个，对方发来一句话，盯着屏幕想了十分钟不知道怎么回。然后打开百度搜"女生说我该怎么做"，出来的全是镰刀——998 的课、2999 的私教、甚至杀猪盘。
>
> **这个项目就是为这件事存在的。** 不教你把妹，不教 PUA，不让你花冤枉钱。就是把一本实战经验写成的书做成 AI 教练，在你需要回复的时候给你参谋。策略、话术、风险提示——像 Code Review 一样审视每一条消息。
>
> **首次使用**：在 Claude Code 中打开此目录，自动进入引导式画像建立。

## 快速开始

### 1. 配置 API Key

在 `.claude/settings.json` 中填入你获得的 API Key：

```json
{
  "mcpServers": {
    "dating-coach-knowledge": {
      "url": "https://正在开发/sse",
      "headers": {
        "x-api-key": "dck_你的API-Key"
      }
    }
  }
}
```

> 💡 **没有 API Key？** 联系管理员获取。API Key 格式为 `dck_` 开头的一串字符。

### 2. 在 Claude Code 中打开此目录

```
claude E:\vscode_project\dating-coach
```

## 使用流程（以 Soul 为例）

### 第一步：交代你自己的情况

首次打开时，Agent 会分 4 轮引导你填写画像——年龄、职业、收入、感情经历、MBTI 性格。**老老实实填，越真实策略越准。**

如果以后有变化，随时说"补充画像：xxx"，Agent 会同步更新。

### 第二步：有女生回复你了

打开 Soul（或其它社交软件），有女生回了你的消息。这时候把两样东西告诉 Agent：

1. **她的个人介绍**——Soul 主页上她写的那些文字（自我介绍、要求、标签等等）
2. **你们的聊天记录**——从第一条到最新一条，复制过来

> Agent 会先做"反向翻译"——她写的每一条要求，翻译成她前任踩过的坑。然后根据你的画像匹配，确定策略方向。

### 第三步：选人，开档案

Agent 分析完后，你说"开个新联系人"，Agent 会在 `contacts/` 下给她建一份独立档案，给她一个编号（如 #001）。

### 第四步：逐条拆招，加上微信

之后每一次——

1. 她发来新消息 → 把她的原话复制给 Agent
2. Agent 分析她在干什么（审查？调情？给绿灯？）
3. Agent 给你 **📱 直接发：** 后面跟着可以直接复制的话术
4. 你复制 → 粘贴到 Soul → 发出去

**每一步都有人帮你看着，直到加到微信。**

## 知识库预览

API 背后的知识库包含 36 章实战内容 + 术语表 + 模式手册。随手摘几个：

### 反向翻译——她说的 ≠ 她想的

| 她说 | 实际含义 |
|------|---------|
| 要专一 / 1v1 | 被出轨过 |
| 要上进 | 前任躺平 / 啃她 |
| 互相花钱花心思 | 被白嫖过 |
| 随时查岗 | 前任撒谎 / 玩消失 |
| 要真诚 / 别骗我 | 被骗过 |

### 术语速查

| 术语 | 含义 |
|------|------|
| 降维打击 | 在条件远低于自己的群体中找目标，成功率极高 |
| 种心锚 | 让对方持续为你做小事，不断提醒她"你为我付出了" |
| 火力压制 | 短时间内高密度多维追求（礼物+情书+偶遇同时上） |
| 与众不同法则 | 对美女的核心策略——所有人都跪舔时，你表现得不屑 |

### 核心框架（节选）

> **投资回报思维**：把追求视为有预算的投资。前期必须投入，做好一部分打水漂的准备。花小钱办大事 > 白嫖 > 无底线砸钱。

> **沉没成本**：让对方投入越多，越难离开——"失去的痛苦永远大于获得时的快乐"。

> **自我认知三问**：行动之前——你有什么能吸引对方？你要什么（1-3-5 年）？你不要什么（底线不退）？

## 为什么订阅 MCP API？

本书的知识可以免费查阅，但如果你想要的不只是"翻书"，而是——

**🎯 个性化，不是通用答案**

你花 998 买的 PUA 课，给所有人的话术一模一样。但你和你兄弟性格不一样——一个内向务实，一个外向感性，你们不可能用同一套话。这个系统会根据你的 MBTI 出最适合你性格的回复——不别扭、不油腻、像你自己说的话。

**💬 实战参谋，不是背台词**

书里写的是一般性原则，但你面对的是具体的人——"她说这句话是什么意思？"、"现在约还是不约？"、"沉默了我该不该追？"——教练系统会读你的画像、调她的档案、分析当前对话，然后给你**可以直接复制发送**的话术，附上设计逻辑。

**🔒 反收割雷达**

市面上最赚钱的不是教你怎么追女生，而是利用你社交经验少、急于求成的弱点收割你。998 的课、2999 的私教、甚至杀猪盘和酒托。教练系统本身就是一本防骗手册——从对方的交友要求反向翻译她的真实意图，识别绿茶套路和消费操纵，让你不花冤枉钱。

**📈 画像持续生长**

不是一个静态工具箱。你每回复一条消息、每结束一段关系、每发现一个新的盲区——教练系统会捕捉、归纳、写入你的画像，从骨架 v1.0 长到血肉 v2.0。下一次自动调用，越用越懂你。
